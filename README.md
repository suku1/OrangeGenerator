# OrangeGenerator
**例のアレ**  

## Requires 
 - TensorFlow
```bash
$ pip install tensorflow
```
 - Pillow
```bash
$ pip install Pillow
```

## Usage  
```bash
$ python Generator.py 生成枚数(未入力の場合は100, 上限1000)
```
